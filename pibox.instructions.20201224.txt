1. Flash OS
	A. Version: Raspberry Pi OS Lite (Release: 2020-12-02)
	
2. Setup headless mode
	A. At the bottom of config.txt, add:
		dtoverlay=dwc2
#		echo "dtoverlay=dwc2,dr_mode=peripheral" | sudo tee -a /boot/config.txt
	B. In cmdline.txt after rootwat add:
		modules-load=dwc2,g_ether
	C. Create file called:
		ssh
	D. [See if this works] Pre-load wpa_supplicant.conf
	ref-01. https://gist.github.com/gbaman/975e2db164b3ca2b51ae11e45e8fd40a
	ref-02. https://stackoverflow.com/questions/44053344/g-multi-mode-mass-storage-ethernet-not-working-on-raspberry-pi-zero-w

3. [Optional] Setup wifi (good backup in case g_ether gets messed up)
	A. SSH into the pi:
		user: pi
		pass: raspberry
	B. Edit wpa_supplicant.conf:
		sudo nano /etc/wpa_supplicant/wpa_supplicant.conf
		a. Add:
			country=US

			network={
					ssid="SSID"
					psk="PASS"
			}
		b. Save + Close
	C. Reboot:
		sudo reboot
	ref-01. https://www.youtube.com/watch?v=xj3MPmJhAPU&feature=youtu.be

4. Create the container file
	A. SSH into the pi
	B. Create binary file:
		sudo dd bs=1M if=/dev/zero of=/piusb.bin count=2048
	C. Format FAT32:
		sudo mkdosfs /piusb.bin -F 32 -I
	D. Create the directory:
		sudo mkdir /mnt/usb_share
	ref-01. https://magpi.raspberrypi.org/articles/pi-zero-w-smart-usb-flash-drive

5. Setup g_multi
	A. Open cmdline.txt:
		sudo nano /boot/cmdline.txt
		a. Delete g_ether
		b. Save + Close
	B. Configure modules:
		sudo nano /etc/modules
		a. Add:
			dwc2
			g_multi
		b. Save + Close
	C. Setup rc.local:
		sudo nano /etc/rc.local
		a. Before exit 0, add:
			sleep 5
			sudo modprobe g_multi file=/piusb.bin stall=0 removable=1 ro=0
		b. Save + Close
	D. Reboot:
		sudo reboot
	ref-01. https://stackoverflow.com/questions/44053344/g-multi-mode-mass-storage-ethernet-not-working-on-raspberry-pi-zero-w
	ref-02. https://gist.github.com/gbaman/50b6cca61dd1c3f88f41#gistcomment-1822387
	ref-03. https://gist.github.com/gbaman/50b6cca61dd1c3f88f41#gistcomment-2850936
	ref-04. https://raspberrypi.stackexchange.com/questions/79699/g-mass-storage-on-rpi-zerow-is-readonly-use-as-usb-stick
	
6. Setup Dropbox Uploader
	A. Follow the instructions at https://pimylifeup.com/raspberry-pi-dropbox/
	ref-01. https://github.com/andreafabrizi/Dropbox-Uploader
	

x. Dropbox Uploader commands:
	unmount from USB, mount to Pi: 
		sudo modprobe -r g_multi; sudo sudo mount /piusb.bin /mnt/usb_share;
	unmount from Pi, mount to USB:
		sudo umount /mnt/usb_share; sudo modprobe g_multi file=/piusb.bin stall=0 removable=1 ro=0;
	
	download from Dropbox:
		Dropbox-Uploader/dropbox_uploader.sh download /usb_share /mnt/
	upload to dropbox:
		 Dropbox-Uploader/dropbox_uploader.sh upload /mnt/usb_share /
